# Asset Credits

## Kenney Assets (CC0 — Public Domain)
- **Isometric Landscape** — terrain tiles (grass, dirt, water, sand, etc.)
- **Isometric Roads** — road tiles, bridges, terrain base tiles
- **Isometric City** — building sprites, urban tiles
- **Tanks** — explosion sprites, tank top-down references
- **Smoke Particles** — smoke and explosion effect frames
- **Particle Pack** — muzzle flash, sparkle effects

All Kenney assets are licensed under **Creative Commons CC0** (public domain).
Source: https://kenney.nl

- Ore/gem terrain tiles
- Selection circle, rally flag effects
- Building construction stages and damage overlays

Generated by `tools/generate_sprites/main.go` using Go's `image` package.
